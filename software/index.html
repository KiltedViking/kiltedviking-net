Introduction

I very much like simple software (made for one or two purposes - not a thousand). Here are some free, and simple, software that I use and recommend. The software is (or was) free to use for personal or educational use. (Please read license agreement to make sure!)

Software categories listed below:

  • Essential - everyday software (e-mail, web browser, etc.)
  • Entertainment - for relaxation
  • Development - for web and software development

Some of the software is for Windows, some for Linux, and some for both: [W] = Windows, [L] = Linux, and [W/L] Windows and Linux.

Essential

  • E-mail: Mozilla Thunderbird [W/L]
  • PDF viewer: Adobe Acrobat Reader [W/L]
  • Web browser: Mozilla Firefox [W/L]

Below are some useful software I believe everybody should have.

  • PDF: Cute PDF - When sending documents (that aren't supposed to be edited by others) one should convert them into PDFs (Portable Document Format)

Entertainment

Some people might say that this software should be listed under Essential above. :-)

  • Snood - a great game one can spend hours playing.

Development

OK, OK, OK... so this software is not all that simple, but it is (or was) free. :-)

Software below is developed as free software (or open source).

  • Apache HTTP Server [W/L] - web server that can be extended with modules or other software.
  • Crimson Editor [W] - simple text editor for HTML, CSS, PHP, SQL, and much, much more. (I used this to create this web site.)

Some of the software below is (or was) free to use for educational purposes, i.e. there might be two or more versions of software (please check before use).

  • MySQL [W/L] - databas management system (DBMS).
  • Oracle [W/L] - databas management system (DBMS).
  • Resin [W/L] - web server with support for Java servlets, JSP, EJB, and more. Can be integrated with web servers like Apache and Microsoft IIS.
