Introduktion

Google erbjuder ett antal verktyg och tjänster till webbutvecklare och på dessa sidor beskriver och visar jag exempel på några som jag använder eller har tittat på, bl.a. de i lista nedan.

  • Google Analytics (GA)- statistik över besök och aktiviteter i webbsidor
  • Google Tag Manager (GTM) - verktyg för att dynamiskt infoga innehåll i webbsidor (bl.a. GA-kod)

Google Analytics (GA)

Tjänst för att samla information om besökare av webbsidor, bl.a. sidvisningar, aktiviteter/händelser.

Jag använder Google Tag Manager för att infoga GA-kod i sidor på min webbplats.

Google Tag Manager (GTM)

Ger möjlighet att dynamiskt infoga innehåll i webbsidor, bl.a.

  • JavaScript-kod för Google Analytics
  • HTML-kod (se notifiering [alert] längst ner på denna sida)

Uppgifter

  • Infoga Google Analytics 4 (GA4) - fånga statistik från besökare på webbplats.
