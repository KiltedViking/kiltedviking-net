Meddelanderutor i JavaScript

HTML-kod:

<p>Du svarade '<span id="spanSvar"></span>' i första dialogrutan.</p>

Kod:

<script type="text/javascript">
  var res = ""; //Variabel för att lagra resultatet från dialogrutor

    //Visa en dialogruta med knapparna OK och Avbryt
  res = confirm("Är du säker?");

  if(res) //Testa om resultatet var OK...
    svar = "OK"
  else //... annars var det Avbryt
    svar = "Avbryt";

  //Hämta referens till element att skriva text till
  var spanSvar = document.getElementById("spanSvar");

  //Skriv ut resultatet
  spanSvar.innerHTML = svar;

</script>

Resultat:

Du svarade '' i första dialogrutan.

För att köra exemplet igen kan du klicka på Uppdatera/Reload i webbläsaren.
