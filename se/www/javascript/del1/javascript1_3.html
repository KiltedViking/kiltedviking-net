Meddelanderutor i JavaScript

HTML-kod:

<p><span id="spanSvar"></span></p>

Kod:

<script type="text/javascript">
  var res = ""; //Variabel för att lagra resultatet från dialogrutor

  //Hämta referens till element att skriva resultat till
  var spanSvar = document.getElementById("spanSvar");

    //Visa en dialogruta för att hämta en sträng från besökaren
  res = prompt("Vad vill du svara?", "startvärde");

    //Bör testa värdet returnerat från metoden prompt() då den returnerar
    // null i fall besökaren klickade på knappen Avbryt.
  if(res == null) //Testa om resultatet var Avbryt...
    spanSvar.innerHTML = "Du tryckte på 'Avbryt' i dialogrutan";
  else //... annars kan vi använda strängen som besökaren fyllde i
    spanSvar.innerHTML = "Du fyllde i '" + res + "' i andra dialogrutan.";
</script>
