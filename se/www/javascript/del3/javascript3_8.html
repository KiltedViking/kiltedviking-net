Objektet location i JavaScript

Kod:

<script type="text/javascript">
  document.write("Serverns adress och domän: <b>" + location.host
    + "</b><br>");
  document.write("Serverns namn: <B>" + location.hostname + "</b><br>");
  document.write("URL till hemsidan: <B>" + location.href + "</b><br>");
  document.write("Sökväg till dokument på server: <b>"
    + location.pathname + "</b><br>");
  document.write("Porten som webbserver använder: <b>" + location.port
    + "</b><br>");
  document.write("Protokoll som används: <b>" + location.protocol + "</b><br>");
</script>
