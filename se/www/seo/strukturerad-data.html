Introduktion

Strukturerad data används för att informera sökmotorer om vilken data som finns i webbsida, t.ex. vad sidan beskriver (företag, evenemang, brödsmulor, etc.). Om data finns i sida så kan den taggas med dess innebörd, annars kan data läggas till som "metadata" (i form av JSON, t.ex.).

Definitioner av stukturera data finns samlade på webbplatsen Schema.org, ett samarbete mellan bl.a. Google, Microsoft, Yahoo och Yandex.
