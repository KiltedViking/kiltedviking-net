Innehåll

På denna sida beskrivs hur man bör utforma HTML för att stilmallar ska fungera bäst.

Det finns några viktiga saker som måste följas för att CSS ska fungera korrekt.

HTML-kod måste vara korrekt

Om inte HTML-koden är korrekt så kan resultatet vara bli oförutsett.

  • Taggar måste ha både öppnande och avslutande tagg. Om tagg saknar innehåll (så som taggarna br och img), så räcker det med att tagg avslutas med ett snedstreck.

Använd attributen class och id i taggar

  • Använd attributet class i HTML-taggar som ska ha samma stil.
  • Använd attributet id i HTML-taggar som är unika i respektive sida (t.ex. "footer", "menu" och "header") och ska ha samma stil.
