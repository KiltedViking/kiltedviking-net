Var JavaScript-kod skrivs

JavaScript-koden kan skrivs (nästan) var som helst i ett HTML-dokument. Men koden måste skrivas inom HTML-taggen SCRIPT. Attributet TYPE i SCRIPT-taggen kan användas för att ange vilket språk som skriptet är skrivet med, i detta fall JavaScript då attributets värde är "text/javascript".

(Texten i denna webbsida har hämtats från underwebben JavaScript på webbplatsen Björns studiewebb.)

Exempel:

<html>
<head>

<script type="text/javascript">
  //T.ex. metoder och globala variabler som
  // anropas från skript nedan
</script>

</head>
<html>
<body>
<!-- HTML-kod -->

<script type="text/javascript">
  //Koden för skriptet
</script>

<!-- Mer HTML-kod -->
</body>
</html>

Dölja JavaScript-koden för äldre webbläsare

Vanligen så brukar man dölja koden för äldre webbläsare genom att "kommentera bort" koden med en HTML-kommentar. Viktigt är (speciellt för Netscape) är att man måste "kommentera bort" den avslutande kommentartaggen m.h.a. JavaScripts kommentar. (Lite omständigt men så fungerar det...)

<html>
<head>

<script type="text/javascript">
<!-- //Dölj för äldre webbläsare
  //... koden ...
// Kommentera bort avslutande tagg-->
</script>
