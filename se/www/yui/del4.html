På denna sida beskrivs man kan söka ut relaterade element, t.ex. syskon-, föräldra- eller barnelement.

Inledning

Utsökningsmetoderna på denna sida används för att hitta relaterade element, t.ex. barn-, syskon- och föräldraelement.

Till början på sida

Metoder

Metoderna nedan utgår från ett element för att välja ut relaterade element.

Fler selektorer
Metod Element som väljs ut
.ancestor(selektor1, [inkludera-själv[, selektor2]]) Förfader som matchar selektor1. Om inkludera-själv är sann så inkluderas element som metod anropas på i sökning. Argumentet selektor2 kan användas för att ange när sökning ska avbrytas.
.ancestors(selektor, [inkludera-själv]) Förfader som matchar selektor1. Om inkludera-själv är sann så inkluderas element som metod anropas på i sökning.
.next([selektor]) Nästa syskon som matchar selektor, eller som skickad funktion returnerar sant för. Om selektor utelämnas väljs nästa syskon.
.previous([selektor]) Föregående syskon som matchar selektor, eller som skickad funktion returnerar sant för. Om selektor utelämnas väljs föregående syskon.
.siblings([selektor]) Alla syskon som matchar selektor, eller som skickad funktion returnerar sant för. Om selektor utelämnas väljs nästa syskon.
Exempel

I exempel nedan så väljs först alla länkar och sen närmaste förälder som är DIV-tagg.

Y.all("a").ancestor("div");
