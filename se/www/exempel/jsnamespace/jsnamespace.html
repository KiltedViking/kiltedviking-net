JavaScript namespace

Detta exempel visar på hur man kan skapa namnutrymme i JavaScript för att förhindra namnkonflikter mellan egen kod och eventuell kod från andra utvecklare.

Klicka på knappar för att anropa funktionen helloWorld() i de två olika namnutrymmen.
