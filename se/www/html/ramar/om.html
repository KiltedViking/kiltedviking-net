Om

Denna hemsida skulle kunna berätta om hur webbplatsen fungerar eller vad organisationen gör.

Observera att man bör använda taggen <BASE> i <HEAD> för att ange var hemsidorna som länkarna refererar till ska visas. Om man inte gör det så kommer hemsidan visas i samma ram som länken. Detta är inte vidare praktiskt om man har en smal menyram till vänster (som denna hemsida...:-). För att visa hemsidorna i ramen med namnet "INFO" (ramen som du läser detta i) så lägger man till följande:

<BASE TARGET="INFO">

Om man vill använda länkar som öppnar en hemsida utan ramar (bra för externa länkar) kan man använda attributet TARGET med värdet "_top", så som Tillbaka-länken i menyn:

<A HREF="../" TARGET="_top">Tillbaka</A>
