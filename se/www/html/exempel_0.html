Att göra: skriv om beskrivningar för mer tidsenliga webbläsare. ;-)

Oroa dig inte om HTML-koden för denna hemsida verkar komplex - det är det denna webbplats är till för att avhjälpa!
Vi kommer att se mer komplex kod än denna. :-)

Gör följande för att se HTML-koden....

...om du använder Microsoft Internet Explorer

  1. Högerklicka någonstans i denna hemsida.
  2. Välj Visa källa från menyn som visas. (För att välja ett alternativ från menyn klickar du med vänster musknapp.)
  3. Anteckningar (eng. Notepad) visas med HTML-koden.
  4. Stäng programmet Anteckningar när du är klar.
  5. Stäng detta fönster (med denna hemsida i). Du kommer då tillbaka till hemsidan med länken till denna hemsida.

...om du använder Netscape Navigator (eller Communicator)

  1. Högerklicka någonstans i denna hemsida.
  2. Välj View source från menyn som visas. (För att välja ett alternativ från menyn klickar du med vänster musknapp.)
  3. Netscapes egna program för att visa HTML-kod (bör) visas.
  4. Stäng programmet/fönstret när du 'r klar.
  5. Stäng detta fönster (med denna hemsida i). Du kommer då tillbaka till hemsidan med länken till denna hemsida.

(Fördelen med Netscape Navigator är att den färgkodar HTML-koden, vilket gör det lättare att läsa koden.
T.ex. blir alla HTML-taggar lila och kommentarer, och litet till, för kursiv stil.)
