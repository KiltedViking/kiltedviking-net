Om listan

Alfabetisk lista med alla taggar (element) i version 5 av HTML-standarden, eller snarare den nyare "levande HTML-standarden" som sen 2004 hanteras av gruppen WHATWG.

Där taggar visas med både en öppnande och avslutande tagg så finns bägge - övriga saknar avslutande tagg. Om avslutande tagg inneslutits mellan hakparenteser ("[" och "]") så är den inte obligatorisk, men man bör dock använda avslutande tagg för att t.ex. stilmallar ska fungera korrekt.

De taggar som har ändrats sen version 4.01 har markerats med:

UTGÅENDE
kommer utgå i framtida version av HTML (eng. deprecated).
UTGÅTT
stöds inte av HTML 5.
NY
tillkommit i HTML 5.

Taggar som utgått i HTML 5

Taggar nedan har tagits bort från HTML 5, men stöd kan finnas kvar i webbläsare.

<acronym> </acronym> UTGÅTT: Används för att skapa en popp-upp beskrivning av akronym/förkortning (RTFM)
<applet> </applet> UTGÅTT: Infogar en Java-applet i hemsidan
<basefont> UTGÅTT: Font för hela dokumentet
<big> </big> UTGÅTT: Stor text
<center> </center> UTGÅTT: Centrerar element
<dir> </dir> UTGÅTT: Lista för korta punkter 
<font> </font> UTGÅTT: Ändrar färg, storlek och typsnitt för text
<frame> UTGÅTT: Ram
<frameset> </frameset> UTGÅTT: Samling med ramar
<isindex> UTGÅTT: Visa textkontroll som besökare kan fylla i för att t.ex. söka 
<noframes> </noframes> UTGÅTT: Text som visas om webbläsare inte kan hantera ramar
<strike> </strike> UTGÅTT: Genomstruken text
<tt> </tt> UTGÅTT: Text med fast teckenbredd 
<u> </u> "UTGÅTT": Understruken text - ska nu användas för oartikulerad annotering, t.ex. som med kinesiska namn
