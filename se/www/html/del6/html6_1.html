Taggen <address>

Nedan visas exempel på två adresser - den första på en rad och den andra delad på flera rader med <br />-tagg. Som exempel visar så går det att använda address-taggen utan varken p- eller div-taggar och att taggen inte infogar en rads avstånd (så som p-taggen).

Taggen <cite>

Frasen "I'll be back" är ett citat i mening nedan. För musen över frasen för att se referensen.

Om vill verka tuff så kan man säga I'll be back, men det kräver nog solglasögon och en hel del muskler.

Taggen <code>

Frasen "var summa = talA + talB;" är ett stycke kod i mening nedan.

JavaScript kod kan se ut enligt följande var summa = talA + talB;.
