Bootstrap Icons

How to add Bootstrap Icons to web page.

Setting up manually

  1. Download CSS file (bootstrap-icons.min.css) and place file in a folder, like /css.
  2. Download fonts (and SVG's) and place font files in a folder fonts located in same folder as CSS file (i.e. with path /css/fonts ).
  3. Add link to CSS file in web page.
  4. Add an i element with class corresponding to wanted icon, like bi-link () and bi-save ().
