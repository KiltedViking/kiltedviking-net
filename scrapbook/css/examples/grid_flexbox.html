Grid and flexbox

Introduction

Grids and flexboxes are created by defining a container and items within.

display: grid;

display: flex;

Default behaviour

Grid

Grids stack on top of each other by default, but can be made to arrange horisontally too.

Alternative direction

Grid

Direction of grid can be changed by defining a template for column and example below uses the repeat() function and passing how many times to repeat and what to repeat (fr = fraction).

grid-template-columns: repeat(3, 1fr);
